    # 对象

    如何定义面向对象和基于对象的问题

    语言和宿主的基础设施有对象来提供，并且JavaScript程序也是一系列相互通信的对象集合

    # 什么是面向对象？

    Object 是一切事物的总称

    # 对象的特性

    1. 要具有唯一标识性: 即使两个完全相同的对象，也并非同一个对象
    2. 要具有状态： 同一个对象可能处于不同的状态之下
    3. 具有行为： 对象的状态可能因为他的行为产生变迁

    # 对象的两类属性

    • 数据属性 value： 就是属性的值 writable： 属性能否被赋值 enumerable： 属性能否被for in 枚举 configurable： 属性能否被删除或者修改

    • 访问器属性 getter：在取属性值是被调用 setter：在设置属性值是被调用 enumerable： 属性能否被for in 枚举 configurable： 属性能否被删除或者修改

    # 什么是前端路由

    • 路由的概念是服务器上，URL和处理函数的映射关系

    • 前端路由是在单页应用中体现小数的是URL和UI也免得映射关系， 即URL变了一起UI更新(无需刷新页面)

    # 如何实现前端路由

    1. 如何改变URL却不引起页面的刷新
    2. 如何检测到URL变化了

    • hash 实现： hash是URL中#后面的那一部分，改变url的hash， 页面不刷新

    • hashchange 可以监听 url 变化

    • history 实现:是HTML提供的一个对象， 其中提供了 pushState 和 replaceState 两个方法可以改变URL且不引起页面刷新

    • popstate 可以监听 pushState 和 replaceState 的 URL变化

    • pushState 和 replaceState 的调用和 a 标签的点击事件是可以被拦截的
